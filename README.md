# KOA API

service for handling authentication and CRUD operation for fellows


## installation

```bash
npm install
```

## requirements

- node 8.x
- mongo 3.2.x w/ ssl support (running in background)

## usage

to run in development mode:
```bash
npm run dev
```

server-side (not for local use):
```bash
npm start
```

to run lint check:
```bash
npm run lint
```
